***** 자동화 검증 설정 방법 *****
1. 자동화 스크립트 폴더 열기
비주얼 스튜디오 > 파일 > 모바일 Btv 폴더 열기


2. 자동화 대상 단말명 입력
비주얼 스튜디오 > mBtv.resource 파일 > Variables의 ${deviceName} 우측에 단말명 입력
ex) ${deviceName}    R5CR8050X6P


3. 앱피움 서버 가동
cmd > appium --base-path=/wd/hub 입력


4. 테스트 실행 (Btv 앱 실행 상태에서 테스트 진행)
비주얼 스튜디오 > 터미널 > 하기 명령어 입력

- Suite 단위 테스트 : robot .\bat.robot
- 개별 테스트 : robot --test "티씨명" .\bat.robot
- 반복 테스트 : for ($i=1; $i -le 20; $i++) { robot --test "티씨명" .\bat.robot }


5. Report 저장
v.s 터미널 > robotmetrics 입력 > 스크립트 폴더 내 저장 확인 ("metrics-날짜" 형식으로 저장됨)

6. Github 업로드 및 삭제 방법
-업로드 방법
https://github.com/NCLQA/mobile-btv 진입 후 Add file > Upload files > 올릴 폴더 드래그하여 드래그 공간에 놓기 > Commit changes
-업로드 삭제 방법
삭제 폴더 및 파일 진입 > Delete directory 또는 Delete file > Commit changes > 팝업 노출 후 Commit changes 선택

***** 자동화 검증 시 수행 방법 *****
1. 단위 테스트 후 Fail 건은 수정 후 반복 테스트로 10번 수행 후 PASS 확인

2. Xpath 값이 정확하지 않으므로 ID 값 우선 적용
